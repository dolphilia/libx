# إنشاء الوثائق

بعد الاطلاع على صفحة البدء السريع، حان الوقت لإضافة صفحات جديدة تضمن أن الروابط السابقة/اللاحقة تعمل بشكل صحيح.

## استخدام النصوص البرمجية

أسهل طريقة هي تشغيل سكربت الإنشاء من جذر المستودع:

```bash
node scripts/create-document.js sample-docs ar v2 guide "إنشاء الوثائق"
```

- ينشئ الملف داخل `apps/sample-docs/src/content/v2/ar/01-guide/`
- يضيف ترويسة frontmatter بالعنوان والوصف
- يحافظ على الترقيم (`02-...`) لضبط ترتيب الشريط الجانبي

## الإنشاء اليدوي

إذا رغبت في الإنشاء يدويًا:

1. أنشئ الملف:
   ```bash
   touch apps/sample-docs/src/content/v2/ar/01-guide/02-creating-documents.mdx
   ```
2. أضف الترويسة أعلاه مع وصف موجز.
3. اكتب المحتوى باستخدام Markdown/MDX مع مراعاة الاتجاه RTL خارج كتل الأكواد.

## نصائح التنظيم

- احتفظ بالملفات المتعلقة بموضوع واحد تحت نفس المجلد مثل `01-guide`.
- استخدم أرقامًا مضاعفة من خانتين (01، 02، 03) كي لا يختل الترتيب عند إضافة صفحات مستقبلية.
- بعد إضافة ملفات جديدة، شغّل `pnpm build:sidebar` لتحديث ملفات JSON المستخدمة في الشريط الجانبي.

بذلك ستحصل على صفحة ثانية تجعل تجربة التنقل بالعربية أكثر واقعية.
